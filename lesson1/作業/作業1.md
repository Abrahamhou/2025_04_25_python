# 標題


# 文字樣式

可以在**評論**的*欄位*和 .md 檔中以 ***粗體、斜體***、~~刪除線~~ 、<sub>下標 </sub> 或 <sup> 上標 </sup>文本表示強調。



# 標注

使用`單反引號`可標注句子中的代碼或`命令`。 反引號中的文本不會被格式化。

The background color is`#0969DA` for light mode and `#000000` for dark mode.
